**HOW TO GET STARTED**

STEP 1: Run to install all required packages

```
npm i
```

STEP 2: Run `cmd` to change the root of sampleTree

**NB** in File src/index.ts uncomment code

```
// import sampleGraph from './utils/sampleGraph';

// changeTreeRoot(sampleGraph, 'D');
```

Then Run

```
npm run dev
```

**100% test coverage**

```
npm run test
```
